## Quick start

Requires [Nerd fonts] (https://github.com/ryanoasis/nerd-fonts)

```bash
  git clone https://github.com/andreystarkov/dotfiles ~/dotfiles
  cd ~/dotfiles
  bash ./setup.sh
```
